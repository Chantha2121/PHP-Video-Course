## 1. ទាញយក

* ចូល 👉 [https://servbay.com](https://servbay.com/)
* ជ្រើស **Download**
* ជ្រើស installer តាម OS របស់អ្នក:
  * **macOS 12+** (Apple Silicon ឬ Intel)
  * **Windows 10+ / Server 2012+**

## 2. ដំឡើង

* macOS:
  * បើក `.dmg`
  * Drag **ServBay** ទៅ **Applications**
* Windows:
  * បើក `.exe`
  * ចុច **Next > Install > Finish**

## 3. បើកប្រើ

* macOS: បើកពី **Applications**
* Windows: បើកពី **Start Menu** (ServBay)
* Allow permission (បើមាន)
* ចុច **Start Services**

## 4. ប្រើ Project

* Folder web root (អាចខុសតាម setting):
  ```
  macOS: /Applications/ServBay/www
  Windows: C:\ServBay\www
  ```
* បើក browser:
  ```
  http://localhost
  ```
